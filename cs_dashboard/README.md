# CS 대시보드

공감센터(help-desk) CS 데이터를 수집·분석하는 내부 대시보드.
매 시간 자동으로 데이터를 수집하고, 일별·주별·월별 통계와 분류별 드릴다운을 제공한다.

---

## 파일 구조

```
cs_dashboard/
├── backend/               # 서버 — 추후 별도 서버에서 운영
│   ├── server.py          # FastAPI 라우트 (API 엔드포인트)
│   ├── db.py              # DB 연결·스키마 초기화
│   ├── helpdesk.py        # help-desk API 클라이언트
│   ├── scheduler.py       # 매 시간 자동 수집 스케줄러
│   ├── cs_dashboard.db    # SQLite DB
│   ├── requirements.txt   # Python 패키지 목록
│   └── .env               # 인증 토큰 (공유 금지)
├── frontend/
│   └── index.html         # 대시보드 UI (단일 파일)
├── .gitignore
└── CLAUDE.md              # 개발 가이드 (AI용)
```

---

## 설치 및 실행

### 1. 패키지 설치

```bash
cd backend
pip install -r requirements.txt
```

### 2. 서버 실행 시 인증

서버 시작 시 터미널에서 help-desk 아이디/비번을 직접 입력한다.
입력한 정보는 메모리에만 유지되며 파일에 저장되지 않는다.

### 3. 서버 실행

```bash
cd backend
uvicorn server:app --host 0.0.0.0 --port 8000
```

브라우저에서 `http://localhost:8000` 접속.

| 드릴다운 클릭 | 실제 call_memo 목록 확인 |

## 데이터 수집

- **소스**: help-desk API (`/issue/issues/`)
- **대상**: `model_type=1009`, `is_complete=true` 조건의 완료 이슈
- **주기**: 매 시간 정각 자동 수집 (서버 실행 중일 때)
- **규모**: 약 800건/일

수집 이력은 `collection_log` 테이블에서 확인할 수 있다.

## 분류 시스템

### 원본 분류 vs 새 분류

DB의 `issues` 테이블에는 분류 컬럼이 두 종류 있다.

| 컬럼 | 설명 |
|------|------|
| `category_main` / `category_sub` | help-desk 시스템 원본값 — **수정 금지** |
| `new_category_main` / `new_category_sub` | call_memo 기반 키워드 재분류 (2026-06 적용) |

원본 분류는 수년 전 내부 업무 흐름 기준으로 만들어진 값으로 실제 고객 문제와 맞지 않는 경우가 많다.
`new_category_*` 컬럼이 실제 분석에 사용하는 분류다.

> 현재 프론트엔드는 아직 원본 분류(`category_main/sub`)를 바라보고 있음. `new_category_*`로 전환 작업 필요.

### 대분류 구성 (8개)

| 대분류 | 소분류 |
|--------|--------|
| 기기·하드웨어 오류 | 충전 불량, 터치·화면 불량, 전원·부팅 오류, 기기 파손, 기기 교체 요청 |
| 네트워크·앱 오류 | 와이파이 오류, 학습 끊김·멈춤, 앱 오류, 화상수업 오류 |
| 해지·유지 상담 | 해지 확정, 해지 방어, 해지금·위약금 문의 |
| 미납·결제 | 미납 관리, 카드 변경, 환불·결제취소 |
| 체험 관련 | 체험 취소·미인지, 중복 신청, 체험 로그인 독려 |
| 교재·물류·배송 | 교재 누락·오배송, 배송 처리, 배송 지연·조정 |
| 계정·서비스 | 개인정보 변경, 이벤트·혜택 문의 |
| 기타 | 내부 이력, 부재 처리, 지면학습, 기기 장기미회수, 교사 상담 요청 |

### 분류 현황 (2026-06 기준, 18,724건)

| 구분 | 건수 | 비율 |
|------|------|------|
| 분류됨 | 12,205건 | 65.2% |
| 충돌→기타 | 274건 | 1.5% |
| 미분류 | 6,245건 | 33.4% |

키워드 규칙 및 충돌 처리 로직은 `CLAUDE.md` 참고.

## API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/stats/hourly` | 시간대별 건수 (일별 뷰) |
| GET | `/api/stats/hourly_range` | 날짜 범위의 시간대별 건수 |
| GET | `/api/stats/daily` | 일별 건수 (주/월별 뷰) |
| GET | `/api/stats/category` | 분류별 건수 |
| GET | `/api/issues` | 상세 목록 (드릴다운) |
| GET | `/api/collection/latest` | 마지막 수집 시각 |

공통 파라미터: `target_date`, `period` (day/week/month), `start_date`, `end_date`
